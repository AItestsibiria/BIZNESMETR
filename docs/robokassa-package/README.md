## Что сделать перед отправкой (5 минут работы)

1. Открыть каждый `.md` файл — заменить ВСЕ placeholder'ы **🔴ВПИШИ_ХХХ🔴** на реальные значения:
   - `🔴ВПИШИ_ИНН🔴` → твой ИНН
   - `🔴ВПИШИ_ОГРНИП🔴` → твой ОГРНИП
   - `🔴ВПИШИ_АДРЕС🔴` → юр. адрес регистрации
   - `🔴ВПИШИ_ТЕЛЕФОН🔴` → контактный телефон
   - `🔴ВПИШИ_ФИО_ПОЛНОСТЬЮ🔴` → Новосёлов Евгений Сергеевич
   - `🔴ВПИШИ_ПАСПОРТ🔴` → серия/номер паспорта (если требует Robokassa)

2. Конвертировать каждый `.md` → `.docx` или `.pdf`:

   **Вариант А (быстро):** Открыть в любом текстовом редакторе → File → Save As → Word Document (.docx)

   **Вариант B (одной командой на Mac/Linux):**
   ```
   for f in 01-oferta 02-privacy 03-refund 04-description 05-contacts; do
     pandoc docs/robokassa-package/$f.md -o docs/robokassa-package/$f.pdf
   done
   ```

   **Вариант C (через Google Docs):** загрузить .md → автоконверсия → Скачать как PDF

3. Залить файлы в кабинет Robokassa → Документы → загрузить каждый под нужную категорию.

4. Дождаться модерации (типично 1-3 рабочих дня).

## Параллельно — заполнить .env на VPS

Чтобы реквизиты автоматически отображались на муzaai.ru/contacts и в footer:

```
ssh root@31.130.148.107 'sed -i "/^LEGAL_/d" /var/www/neurohub/.env && cat >> /var/www/neurohub/.env <<EOF
LEGAL_ENTITY_NAME=ИП Новосёлов Е.С.
LEGAL_ENTITY_FULL_NAME=Индивидуальный предприниматель Новосёлов Евгений Сергеевич
LEGAL_INN=🔴ВПИШИ_ИНН🔴
LEGAL_OGRN=🔴ВПИШИ_ОГРНИП🔴
LEGAL_ADDRESS=🔴ВПИШИ_АДРЕС🔴
LEGAL_PHONE=🔴ВПИШИ_ТЕЛЕФОН🔴
LEGAL_EMAIL=hello@muzaai.ru
LEGAL_SNO=npd
LEGAL_DEFAULT_TAX=none
EOF
chmod 600 /var/www/neurohub/.env && pm2 restart neurohub --update-env'
```

После этого https://muzaai.ru/contacts покажет реальные ИНН/ОГРН — Robokassa проверит site footer и увидит твои реквизиты.
